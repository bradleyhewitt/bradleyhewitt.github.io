My Work

In collaboration with Nahum Arredondo, Alex Liang, and Justin Scabarozi — Fake PolyLearn Password Phisher

The assignment for this project was to create a way to "phish" our professor, and gain his password. We attempted it by developing a proxy website that mirrors PolyLearn's login page. The login credentials would then be sent to a text file on our server. I no longer have access to the password collection file anyways, but this should not be used maliciously against other students.
